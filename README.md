# AI-Powered CCTV Surveillance & Multi-Camera Monitoring System

An intelligent AI-based surveillance system designed for real-time CCTV monitoring, object detection, motion tracking, and multi-camera security analysis using Computer Vision and Deep Learning techniques.

---

## 📌 Project Overview

This project provides a smart surveillance solution capable of monitoring multiple CCTV camera feeds simultaneously. The system uses Artificial Intelligence and Computer Vision to detect objects, identify suspicious activities, track motion, and generate real-time security alerts.

The project is designed to improve traditional surveillance systems by introducing automation, intelligent monitoring, and enhanced security analysis.

- 🎥 Real-time CCTV camera monitoring
